# Avatar Setup Instructions

## Generating Avatar Images

To generate the 12 avatar images needed for user profiles:

### Method 1: Using Browser (Recommended)

1. Open the file `public/generate-avatars.html` in your web browser
2. The browser will automatically download 12 avatar images (avatar1.png through avatar12.png)
3. The images will appear in your Downloads folder
4. Move all 12 images to the `public/` directory of the project

### Method 2: Manual Creation

If the automatic generation doesn't work, you can create simple avatar images manually:

1. Create 12 PNG images (200x200 pixels each)
2. Name them avatar1.png through avatar12.png
3. Use different colors/emojis for variety
4. Place them in the `public/` directory

### Method 3: Use Placeholder

If you want to skip this step temporarily:
- The app will fallback to avatar1.png for all users
- You can add the images later

## Avatar Details

Each avatar is:
- 200x200 pixels
- PNG format
- Gradient background with emoji
- Stored in the `public/` directory
- Referenced as `/avatar1.png`, `/avatar2.png`, etc.
